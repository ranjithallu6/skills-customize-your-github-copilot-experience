# 📘 Assignment: Building REST APIs with FastAPI

## 🎯 Objective

Learn how to build RESTful APIs using the FastAPI framework in Python. Practice designing endpoints, handling requests, and returning JSON responses.

## 📝 Tasks

### 🛠️ Basic API Setup

#### Description
Set up a FastAPI project and create a simple endpoint that returns a welcome message.

#### Requirements
Completed program should:
- Use FastAPI to create an API
- Have a `/` endpoint that returns a JSON welcome message

### 🛠️ CRUD Operations

#### Description
Add endpoints to perform Create, Read, Update, and Delete operations for a simple resource (e.g., items).

#### Requirements
Completed program should:
- Support POST to create an item
- Support GET to list all items
- Support PUT to update an item
- Support DELETE to remove an item
- Return appropriate JSON responses

#### Example
```json
{
  "id": 1,
  "name": "Sample Item"
}
```
